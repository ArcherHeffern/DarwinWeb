# About
Frontend for Darwin Auto Grading System

# Setup
## Install Dependencies
```bash
npm install
```

## Modify Deploy URL
Modify DEPLOY_API in src/constants/constants.ts
Find your IP Address with `ifconfig`

# Running
## Run Dev
``` bash
npm run dev
```

## Run Prod
``` bash
npm run prod
```
